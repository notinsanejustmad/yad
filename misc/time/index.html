TIME!

Here's a collection of bad or interesting timetelling:

Madtime!

This is a weird time notation i made, based on my subjective opinion of what values are more important. Try finding out what's going on!
Solution: First day, then hour + minute. After that: Month, year, century and finally seconds.

Madtime 2.0!

Same as above, but a friend wanted me to display AM/PM before the rest.

Hex time!

Hexadecimal time, where the first two digits mark the day (and that's why the red channel only goes up to decimal 31), the rest marks the hour. Everything is displayed as a hex colour in the page background.

Hex time 2.0!

Improved version of the above. The first two digits mark how far we are into the year.

Hex time 3.0!

Improved version of the above. If you want the same time to pass between each colour change you'll have to suffer under horrible colour combinations.

Unary!

Normal clock but 0 = |, 1 = ||, 2 = ||| and so on.

Metric time

This one is very cool! The first text field changes how many hours a day has, the second one how many minutes an hour has, and the last one changes the number of seconds in a minute!
